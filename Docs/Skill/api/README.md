This directory contains the detailed technical reference for all public members of the `LightningArc.Utils` library, organized by namespace.

*   **[LightningArc.Utils.Abstractions](LightningArc.Utils.Abstractions/README.md)**: Common domain value objects.
*   **[LightningArc.Results.AspNetCore](LightningArc.Results.AspNetCore/README.md)**: ASP.NET Core integration for Results.
*   **[LightningArc.CORS.AspNetCore](LightningArc.CORS.AspNetCore/README.md)**: Predefined CORS policies.
*   **[LightningArc.OpenAPI.AspNetCore](LightningArc.OpenAPI.AspNetCore/README.md)**: OpenAPI schema transformers.
*   **[LightningArc.Utils.Data.Abstractions](LightningArc.Utils.Data.Abstractions/README.md)**: Repository and Unit of Work abstractions.
*   **[LightningArc.Utils.Data.ADO](LightningArc.Utils.Data.ADO/README.md)**: ADO.NET and Dapper specific implementations.
