*   **[LightningArc.Utils.AspNet.Results](LightningArc.Utils.AspNet.Results/README.md)**: ASP.NET Core integration for Results.
*   **[LightningArc.Utils.AspNet.CORS](LightningArc.Utils.AspNet.CORS/README.md)**: Predefined CORS policies.
*   **[LightningArc.Utils.AspNet.OpenAPI](LightningArc.Utils.AspNet.OpenAPI/README.md)**: OpenAPI schema transformers.
*   **[LightningArc.Utils.Data.Abstractions](LightningArc.Utils.Data.Abstractions/README.md)**: Repository and Unit of Work abstractions.
*   **[LightningArc.Utils.Data.ADO](LightningArc.Utils.Data.ADO/README.md)**: ADO.NET and Dapper specific implementations.
