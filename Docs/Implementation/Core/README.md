# Core Layer Implementation

The Core layer contains the domain-agnostic logic that can be used in any .NET application.

## Projects

*   **[Utils.Abstractions](Utils.Abstractions/README.md)**: Base types and Value Objects.
*   **[Utils.Results](Utils.Results/README.md)**: The core Result pattern engine.
*   **[Utils.Json](Utils.Json/README.md)**: Serialization adapters.
